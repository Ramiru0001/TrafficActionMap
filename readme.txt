こちらは事故を予測するプロジェクトです。
現在は2020~2023年のデータで学習したモデルを使用しています。
事故データは警察庁が公開しているものを使用しています。


学習するデータを変更したい場合はMachineLearning.pyの
　# 事故データの読み込み　で読み込んでいるファイルを変更して実行してください。


実行ファイル

project.bat
事故予測プロジェクトのサーバー立ち上げ。
ブラウザで「http://127.0.0.1:5000」にアクセスしてください。

地図上をクリックしてマーカーを設置すると自動で現在のその場所の天気が入力され、リスクマップが表示されます。
手動で天気等の条件を変更した場合は「リスクマップを表示」を押すとリスクマップが表示されます。



機械学習用ファイル

MachineLearning.bat
機械学習する際に実行。読み込むファイルを変更したい場合実行する。
poolのコア数を、実行するPCのコア数に変更してから実行してね。

※注意※メモリを最大32GB程度使用するため、64GB以上のPCを使用することを推奨。
天気情報をネットから引っ張ってくるので、ネット環境必須。
ネットにアクセスできないとalertが出ます。



出典:交通事故統計情報のオープンデータ（警察庁ウェブサイト　）
